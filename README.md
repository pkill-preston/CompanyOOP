# CompanyOOP

A simple company management system built to practice Object-Oriented Programming (OOP) concepts in C#.

## Purpose

This project was created as a hands-on way to learn and apply:

- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Constructors
- Validation

The goal is educational rather than production-ready software.

## Features

- Employee management
- Specialized employee types (e.g. Developer)
- Custom validation logic
- Encapsulated domain models
- OOP-focused architecture

## Project Structure

```text
CompanyOOP/
├── Models/

│
├── Validators/
│   ├── AgeValidator.cs
│   ├── NameValidator.cs
│   └── CompanyValidator.cs
│
├── Program.cs
└── companyOOP.csproj
```

## Getting Started

### Prerequisites

- .NET SDK 10.0 or newer

### Run

```bash
dotnet run
```

### Build

```bash
dotnet build
```

## Learning Objectives

This project is intended to explore:

- Designing classes with responsibilities
- Separating validation from models
- Using inheritance to model real-world relationships
- Applying abstraction through base classes
- Understanding object interactions
